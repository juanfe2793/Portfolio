## Case Studies

### [01 — Cloud Native API Gateway](/docs/case-studies/api-gateway)

Architecting a Domain Gateway Infrastructure using Kong at 3M+ RPS and 99.999% availability, with Terraform modules that became the organizational standard.

### [02 — Twilio-wide DNS Modernization](/docs/case-studies/dns-modernization)

Designing a centralized, agnostic DNS boundary for 100+ hosted zones across a multi-region Kubernetes platform — eliminating manual DNS operations entirely.

### [03 — Automated Observability & Load Testing](/docs/case-studies/observability-load-testing)

Shifting platform performance testing left with OpenTelemetry and k6 as CI/CD gates, achieving 100% observability coverage for critical services.
